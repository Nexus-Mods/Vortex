# Agent Instructions

Use `pnpm run` for repo commands. After code changes, run `build`, `test`, `lint`, and `format` on the affected package.

## Conditional Context

- For repo navigation or code search: read `AGENTS-DIRECTORIES.md`
- For testing: read `AGENTS-TESTING.md`
- For frontend/UI work (React/TypeScript renderer): read `AGENTS-FRONTEND.md`
