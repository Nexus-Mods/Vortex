    **File 1: `src/renderer/webpack.config.cjs`** (per D-03, D-04)

    In the `resolve` block (currently lines 47-50), add a platform-conditional alias. The `path` module is already imported at line 7. Add the alias spread AFTER the `extensions` array:

    ```javascript
    resolve: {
        plugins: [new TsconfigPathsPlugin()],
        extensions: [".js", ".jsx", ".ts", ".tsx", ".json"],
        ...(process.platform === "linux" && {
            alias: {
                "winapi-bindings": path.resolve(
                    __dirname,
                    "src",
                    "util",
                    "winapi-shim.ts",
                ),
            },
        }),
    },
    ```

    The spread evaluates at build time. On Windows, `process.platform === "linux"` is `false`, so the spread is `...(false && {...})` which spreads nothing. On Linux, it adds the `alias` key.
